### t2a: text to audio 

asks gpt3.5turbo and produce an audio response. 

4 ways of using it:

 * **t2a** file.txt
 * **t2a** "your text here"
 * **t2a** and a prompt will appear asking you to input text
 * create a .txt file at /dev/shm/file.txt and then run **t2a**
